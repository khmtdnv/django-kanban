Task Manager — REST API сервис для управления проектами и задачами с гибкой методологией Канбан , продвинутой системой аналитики и многоканальными уведомлениями. Проект построен на классической Django-архитектуре с упором на профилирование, оптимизацию запросов и мониторинг.  

Стек технологий: Python, Django, Django REST Framework (DRF), PostgreSQL, Redis, Celery, Docker / Docker Compose (Multistage) , Prometheus, Grafana, Sentry , OpenAPI / Swagger (drf_yasg) , Poetry , Pytest.
